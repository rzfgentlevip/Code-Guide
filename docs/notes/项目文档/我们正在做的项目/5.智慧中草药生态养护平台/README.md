## 项目简介<Badge type="tip" text="v2.0.4_release" />

在中草药种植领域，杂草的疯狂生长对中草药的质量和产量造成了严重影响。为了提高中草药的种植效率和质量，我们开发了一款智慧中草药生态养护平台，旨在通过智能化手段实现对中草药种植环境的全面监控和管理。
