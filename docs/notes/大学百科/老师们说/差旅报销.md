## 应用场景

### ::https://2025.jsjds.com.cn/assets/logo2-1fc42965.png:: 中国计算机设计大赛差旅经费报销

在省赛前，学院负责垫付，直接公对公转账。如入围国赛，需要准备用于报销的材料：

- 登机牌
- 机票的电子发票
- 比赛期间的所有消费记录截图，包括吃、住、行

:::tip 消费
- 交通费上限为80元/人/天，如果当天实际交通费发生大于80，可以凭票报销；
- 每天餐补是100元
:::
